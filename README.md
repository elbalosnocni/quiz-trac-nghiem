# HỆ THỐNG THI NỘI BỘ – BẢN FINAL

## Cách dùng nhanh
1. Upload thư mục này lên GitHub (repo Public)
2. Dán URL Google Apps Script:
   - QUESTION_API_URL: lấy câu hỏi từ Excel/Google Sheets
   - SCRIPT_URL: nhận kết quả thi
3. Bật GitHub Pages (main / root)
4. Gửi link cho nhân viên thi

## File đính kèm
- index.html : file thi chính
- logo.png   : logo công ty (đổi logo của bạn)
- questions.xlsx : file mẫu câu hỏi
